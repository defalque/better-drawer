Drawer

Main features of the library.

Default

The most basic setup for a drawer.

@if (openDrawer()) {

Side drawer

Set the direction input to "right" or "left" to change the position of the drawer.

@if (openSideDrawer()) {

Nested drawers

You can nest drawers inside other drawers to create a stack of drawers. It works with any direction.

}
